# Edit this file with your actual 10 passwords, one per line

To add your 10 real passwords:

1. Edit the passwords file:
   nano config/passwords.txt

2. Replace the 3 example passwords with your 10 real ones

3. Save and exit (Ctrl+X, then Y, then Enter)

4. Verify it has 10 lines:
   wc -l config/passwords.txt
   
5. Run the cracker:
   make crack
